Our Project

Developing a full blown Progressive Web Application (PWA) that includes features like connectivity independence, app like interface, push notification, self updates etc.

Goal 1: Be able to incorporate and use phone's camera hardware into our PWA

Goal 2: Gain access to camera roll and be able to upload pictures to pull text

Goal 3: Pull text from photo and copy to clipboard

Goal 4: Be able to edit the resulting text
